Principais tipos primitivos

    int → 7, -4, 0, 9875
    float → 4.5, 0.076, -15.1, 7.0
    bool → True, False
    str → 'Olá', '7.5', ''

var.isnumeric()
> Se não definirmos o tipo primitivo antes do input, ele será str. Porém, com
esse método ele vai dizer se é possível converter esse valor que digitei em
um número com o tipo primitivo int antes dele
